# 🇻🇪 API de Lugares de Venezuela - PHP + MVC + PDO + MySQL

Esta API proporciona acceso a información estructurada sobre lugares turísticos, históricos y representativos de Venezuela. Está construida en PHP usando el patrón de arquitectura MVC, Programación Orientada a Objetos (POO), conexión a base de datos con PDO y almacenamiento en MySQL.

## Características

- 🌍 Listado de lugares por estado, tipo o nombre
- 🔍 Búsqueda de lugares específicos
- ➕ Añadir, editar y eliminar lugares (CRUD)
- 🧱 Estructura limpia y escalable con MVC
- 🛡️ Validaciones básicas de entrada y manejo de errores
- 📦 Listo para integrarse con frontend o apps móviles

## Tecnologías

- PHP 8+
- MySQL
- PDO (PHP Data Objects)
- MVC (Modelo Vista Controlador)
- JSON como formato de respuesta

## Instalación

1. Clona el repositorio
2. Importa el archivo `lugares.sql` en tu servidor MySQL
3. Configura la conexión en `config/database.php`
4. Ejecuta en tu servidor local y empieza a consumir la API
